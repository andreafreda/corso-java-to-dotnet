# 1.6 Tooling: Visual Studio, VS Code, Rider

> **Analogia Java**: equivalente di IntelliJ IDEA / Eclipse — ma con qualche sorpresa

---

## Il cambio di IDE: parliamoci chiaro

Cambiare linguaggio è una cosa. Cambiare IDE è un'altra — e per molti sviluppatori è quella che fa più male. Se hai passato anni a ragionare con IntelliJ IDEA o Eclipse sotto le dita, le scorciatoie sono muscolari, non mentali. Aprire un editor diverso significa ricominciare da zero su cose che davi per scontate.

Detto questo: se vieni da Java, il tuo livello di trauma dipende molto da cosa hai usato finora.

- **Hai usato VS Code per JavaScript, Python o altro?** Bene, sei già a casa. VS Code per .NET funziona esattamente come lo conosci, con qualche estensione in più.
- **Hai usato Visual Studio in passato** (anche solo per curiosità)? Il salto è gestibile.
- **Hai sempre e solo usato IntelliJ o Eclipse?** Mettiti comodo — ci vuole un po' di adattamento, ma non è la fine del mondo.

---

## I tre strumenti sul tavolo

### Visual Studio

L'IDE storico di Microsoft, disponibile solo su Windows (e Mac, ma la versione Mac è stata ufficialmente dismessa nel 2024). È enorme, potente, e per anni è stato *l'unico* modo serio di sviluppare in .NET.

Oggi esiste in tre edizioni:
- **Community** — gratuita per uso personale, open source e piccoli team
- **Professional** e **Enterprise** — a pagamento, per uso commerciale

Per il nostro corso non lo useremo come strumento principale. È ottimo se sei su Windows e lavori su progetti molto grandi, ma per imparare .NET da zero è più potente di quanto serve — e rallenta il processo.

---

### Visual Studio Code

Un editor leggero, cross-platform, open source e gratuito. Non è un IDE a tutti gli effetti come IntelliJ, ma con le estensioni giuste diventa uno strumento molto capace per sviluppare in .NET.

Le estensioni che ti servono:
- **C# Dev Kit** (Microsoft) — IntelliSense, debug, navigazione del codice
- **REST Client** — per testare le API direttamente dall'editor con file `.http`

**È lo strumento che useremo in questo corso.** I motivi sono pratici:
- È gratuito senza asterischi
- È cross-platform — funziona uguale su Windows, Linux e macOS
- Si integra molto bene con gli agenti AI (GitHub Copilot, Claude Code e altri)
- È quello che troverai configurato nella maggior parte dei tutorial e dei progetti open source .NET moderni

---

### Rider

Qui le cose si fanno interessanti — e un po' malinconiche.

Rider è l'IDE .NET di JetBrains, la stessa azienda che fa IntelliJ IDEA. E si vede: l'interfaccia, la navigazione, il refactoring, la gestione dei progetti — tutto ha quella sensazione familiare che chi viene da IntelliJ riconosce immediatamente. C'è persino la possibilità di installare il **keymap di Eclipse**, se sei uno di quelli (nessun giudizio, io ho installato le shortcut di eclipse anche sulla mia lavatrice).

Il problema è uno solo: **non è gratuito per uso commerciale**. Richiede una licenza JetBrains, come IntelliJ IDEA Ultimate. Esiste una versione gratuita per uso non commerciale, ma non appena apri un progetto per un cliente o per la tua azienda, hai bisogno della licenza.

Quindi, per quanto il cuore propenda per Rider — soprattutto se vieni da IntelliJ e vorresti sentirti subito a casa — la scelta consigliata rimane **VS Code**. È gratuito, funziona bene, e non ti metterà mai in una situazione scomoda sul fronte licenze.

Se le finalità non sono commerciali, ti risulta traumatico il passaggio a VSCode e desideri, per un primo momento, fuori dal pc aziendale, imparare il linguaggio allora benvenuto in zona Xander.

---

## In sintesi

| | Visual Studio | VS Code | Rider |
|---|---|---|---|
| Costo | Gratuito (Community) / A pagamento | Gratuito | A pagamento (uso commerciale) |
| Piattaforme | Windows | Windows, Linux, macOS | Windows, Linux, macOS |
| Feeling Java | Lontano da IntelliJ | Neutro | Identico a IntelliJ |
| Integrazione AI | Discreta | Ottima | Buona |
| Consigliato per il corso | No | ✅ Sì | No (licenza) |

La scelta è VS Code — non perché sia il migliore in assoluto, ma perché è gratuito, cross-platform, e ti porta lontano senza intoppi. Se un giorno la tua azienda ha una licenza JetBrains attiva, passa pure a Rider: ti sentirai subito a casa.
